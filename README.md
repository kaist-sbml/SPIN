# SPIN (Stirring Parameter Intelligent Navigator) / (Smart Parameter Intervention Navigator) 
* Note: This source code was developed in Linux, and has been tested in Ubuntu 20.04 with Python 3.8 and CUDA version 11.7

## Source 

1. Clone the repository 
```
git clone https://github.com/kaist-sbml/SPIN.git
```

2. Create and activate a conda environment
```
conda env create -f environment.yml
conda activate bdo_env
```
